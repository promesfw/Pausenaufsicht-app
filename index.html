🔑 GitHub Token Setup

Für automatisches Speichern benötigen Sie einen GitHub Personal Access Token:

So erstellen Sie einen GitHub Token:

  1. Gehen Sie zu GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Klicken Sie auf "Generate new token (classic)"
  3. Geben Sie eine Beschreibung ein (z.B. "Pausenaufsicht App")
  4. Wählen Sie Scope "repo" (Full control of private repositories)
  5. Klicken Sie "Generate token" und kopieren Sie den Token hierher
